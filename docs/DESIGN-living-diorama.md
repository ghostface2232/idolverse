# 게임 진행 화면 디자인 방향 — "살아있는 디오라마"

> **작성일**: 2026-07-14
> **상태**: 설계 제안 (구현 착수 전)
> **관련 문서**: `docs/REVIEW-2026-07-13.md` (종합 리뷰 — 이 문서는 그 로드맵의 "게임 경험" 축을 구체화한 것)

## 1. 문제 정의

창단 플로우는 완성처럼 느껴지고 게임 진행 화면(대시보드)은 그렇지 않다. 격차의 본질은 그래픽 품질이 아니라 다음 네 가지다.

| 창단 플로우가 잘하는 것 | 대시보드의 현재 상태 |
|---|---|
| **기대 → 선택 → 즉시 피드백** 아크가 화면마다 있음 (예산 슬라이더 → 능력치 밴드 즉시 반응) | 선택(카드 3장)과 결과(WeekReport) 사이의 인과가 보이지 않음 |
| **전용 비주얼 에셋**이 선택에 얼굴을 줌 (투자사 로고, 스태프 스프라이트, 시설 일러스트) | Phaser 뷰는 픽셀 사각형, 탭 아이콘은 알파벳 |
| **단계 구조(StepIndicator)** — 지금 어디쯤인지 항상 보임 | 무엇을 향해 가는지(데뷔 조건, 투자사 KPI) 표시 요소 0 |
| **선택의 무게를 미리 보여줌** (스카우트 예산별 후보 품질 미리보기) | 카드 효과 미리보기 없음, 진행 버튼은 즉발 → 결과 모달 |

즉 고쳐야 할 것은 **"선택→결과 루프의 가시성"과 "리듬(연출 아크)"**이다.

## 2. 설계 원칙

1. **매주가 하나의 에피소드다** — `계획(카드) → 실행(시간의 흐름) → 결과(결산) → 다음 주 훅`의 4박자 아크.
2. **모든 숫자 변화는 눈에 보여야 한다** — 돈·스탯·팬덤이 연출 없이 바뀌는 일이 없도록.
3. **폴리시는 콘텐츠를 못 이긴다** — 연출 작업은 코어 루프 개통(앨범/데뷔/프로모션 배선)과 병행하되 앞지르지 않는다.
4. **화면은 멈추지 않는다, 시뮬레이션만 멈춘다** — 카이로소프트의 착시 구조를 그대로 채용한다 (아래).

## 3. 핵심 아키텍처: 로직은 턴제, 연속성은 표현 계층의 환상

카이로소프트 게임의 "살아있는" 감각을 분해하면: **게임 로직은 이산적인 틱인데, 캐릭터가 틱 사이에도 계속 움직이기 때문에** 연속으로 느껴진다. 이를 현재 아키텍처의 강점(순수 함수 `processWeek`, 시드 결정론, 세이브 안정성)을 지키며 가져온다.

```
[게임 로직]   주 단위 이산 처리 (지금 그대로. processWeek 1회 호출 = 1주)
[표현 계층]   항상 흐른다 — Phaser가 두 가지를 재생:
              ① 앰비언트: 로직과 무관한 코스메틱 연기 (배회, 이모트, 잡담)
              ② 전환 연출: processWeek가 계산해 둔 결과를 요일 축에 풀어 재생
```

**불변 규칙**: 보이는 우연은 전부 연기이고, 실제 상태 변화는 전부 `processWeek` 안에서만 일어난다. 이 선을 넘는 순간(예: 배회 중 "우연히 마주쳐서" 케미가 변함) 결정론과 테스트 가능성이 무너진다. Phaser에는 게임 로직을 절대 넣지 않는다.

## 4. 화면 상태 모델 — 3가지 상태, 모두에서 세계는 살아 있다

### 상태 A: 대기 (주 시작, 결정 미완료)
게임 시간은 멈춰 있지만 화면은 멈추지 않는다.
- 멤버들이 현재 활동 방에서 루프 행동 (연습실: 안무 반복 + ♪ / 숙소: 휴식 + 💤 / 이따금 방 사이 이동).
- **상태 노출 이모트**: 스트레스 높음 💢, 저케미 쌍 조우 ⚡, 저만족 시무룩 — 결정 전에 화면만 봐도 "이번 주 뭘 해야 할지" 읽히는 것이 이 상태의 존재 이유.
- 결정 카드의 인월드 진입점: 매니저 스프라이트가 사무실에서 나와 말풍선 `!` → 탭하면 카드 시트. 하단 "결정 N건" 바는 접근성 보조로 유지.

### 상태 B: 흐름 (주 넘김)
"다음 주 진행"을 누르면 모달이 아니라 **같은 화면에서 시간이 흐른다**.
- 상단 요일 티커(월→일, 총 2.5~4초, **탭 홀드로 즉시 완료**), 멤버들이 요일별 스케줄 소화.
- 성장 시 `+0.8 보컬` 플로팅 텍스트, 지출/수입은 요일에 걸친 잔액 카운트다운/업, 예능 멤버는 화면 밖으로 나갔다 복귀.
- **이벤트는 해당 요일에 그 자리에서 발생**: 흐름 일시정지 + 당사자 줌 + 말풍선(💔/🩹/📞) → 탭하면 선택 모달. 사건이 세계 안의 시간과 장소를 갖는다.
- 요일 배치는 시드에서 결정론적으로 파생 (`processWeek`가 리포트 항목에 `day: 1~7` 메타데이터 부여 — 로직 변경이 아니라 메타데이터 추가).

### 상태 C: 결산 (비차단)
- WeekReport를 전면 모달에서 강등: 하단 "주간 결산" 배너 슬라이드 인(등급 스탬프 + 헤드라인 1줄) → 탭하면 상세, 무시하면 3초 후 신문 아이콘 📰으로 접힘.
- **큰 주만 자동 전체 화면**: 수상, 데뷔, 멤버 이탈, 파산 경고. "평범한 주는 흐르고, 특별한 주만 멈춘다"가 리듬의 핵심.
- 결산이 접히면 자동으로 상태 A로 — 매니저가 다시 걸어 나오며 루프가 이어진다.

## 5. 연속 다주 진행(카이로소프트 모드)으로 가는 계단

위 구조가 자리 잡은 뒤의 확장. 전제 조건 순서대로:

1. **카드 기본 선택지**: 연속 모드에서는 매니저가 기본안을 선택하며, 매니저 능력에 따라 판단의 질이 달라진다. GUIDE.md의 "매니저가 세부 일정을 맡는다"는 설계와 맞물려 매니저 스탯에 실질적인 가치를 부여한다.
2. **정지 규칙**: 위기 카드(스캔들/부상/투자사/사기 저하)와 선택형 이벤트는 무조건 정지. 일반 카드만 위임 가능.
3. **배속(1x/2x/3x)은 마지막**: 연속 모드 안정 후. 수동 주 넘김의 체감을 먼저 완성해야 "위임할 가치가 있는 결정"이라는 인식이 생긴다.

## 6. 보조 시스템

- **WeekReport 결산 격상**: 헤드라인 모먼트(이번 주 최대 사건 1개), 선택→결과 대조(고른 카드와 실제 효과 병기), 멤버별 스탯 before→after, 재정 카테고리 breakdown, 주간 등급 스탬프(S~D), 다음 주 훅(라이벌 컴백 예고, 투자사 D-day).
- **목표 스트립**: `🎯 데뷔까지 스탯 평균 55/70 · 투자사 점검 D-9` — StepIndicator의 인게임 버전.
- **팬덤 4축 미니 게이지 + 뉴스 티커**: 세계가 살아 움직인다는 감각의 최소 구현.
- **카드 경험**: 효과를 아이콘+강도(▲▲/▲)로 표기(정확한 수치 공개와 완전 은폐의 중간), 매니저 코멘트(능력 비례 힌트 정확도), 위기 카드 시각 위계, 딜링 연출.
- **사운드/주스**: howler SFX(버튼/카드/주간 흐름/이벤트 스팅/등급 스탬프), 계절당 BGM 1루프, 돈 카운트업, 토스트, 스캔들 화면 흔들림, 수상 콘페티.

## 7. 유의점 (이 방향의 함정)

- **연출 피로가 최대 리스크**: 52주 × 3초 = 연출만 2.6분/년. ① 탭 홀드 즉시 완료 ② 무사건 주는 1.2초 단축 ③ 설정에서 연출 길이 조절. 스킵해도 정보 손실이 없도록 결산에 전부 남는 것이 전제.
- **흐름과 강제 결정의 충돌**: 이벤트 모달이 흐름을 자주 끊으면 턴제보다 답답해진다. 동시 다발 이벤트는 요일 분산, 무선택 이벤트는 정지 없이 말풍선+토스트로만.
- **360px 가독성**: 플로팅 텍스트/이모트 동시 표시 상한(3개)과 우선순위(부상 > 성장 > 코스메틱).
- **표면 정리(아이콘/한국어 통일)는 마지막**: 화면 구조가 바뀌므로 일괄 처리가 재작업을 줄인다.

## 8. 구현 로드맵

| 단계 | 내용 | 비고 |
|---|---|---|
| **1. 결과 가시화** | WeekReport 구조화(멤버별 스탯 before→after, 선택→결과 대조, 헤드라인, 재정 breakdown), 돈 카운트업 | 모든 연출의 원재료 — 최우선 |
| **2a. 앰비언트 레이어** | 대기 상태 배회/이모트/상태 연기 + 멤버 탭→상세 | 에셋 없이 현 사각형으로 프로토타입 가능 |
| **2b. 이음새 없는 주 넘김** | 요일 흐름 + 인월드 이벤트 + 결산 배너화(모달 강등) | processWeek에 요일 메타데이터 추가 |
| **3. 목표/정보 위젯** | 목표 스트립(투자사 KPI·데뷔 조건), 팬덤 4축 게이지, 뉴스 티커 | 2와 병행 가능 |
| **4. 에셋 패스** | 멤버 스프라이트, 계절 테마, 시설 레벨 반영, 사무실에 스태프 배치 | 에셋 제작이 병목 |
| **5. 카드 경험 + BGM** | 효과 강도 표기, 매니저 코멘트, 위기 위계, 딜링 연출, SFX/BGM | |
| **6. 매니저 위임 + 연속 진행** | 기본 선택지 + 정지 규칙 + (이후) 배속 | 5까지 완료 후 |
| **7. 콘텐츠 결합 연출** | 앨범 제작 몽타주, 컴백 주간 스페셜, 시상식 시퀀스 | REVIEW Phase 1 배선 후 |

**코어 루프 개통(REVIEW Phase 1)과의 교차 진행 권장**: 1 → (앨범·차트 배선) → 2a·2b → (승패 조건) → 3·4·5 → 6 → 7. 연출만 먼저 완성하면 "예쁘게 포장된 빈 상자", 콘텐츠만 먼저면 체감 문제가 그대로 남는다.
